机器学习（ML，machine learning）概述：

一、学习：
动物从大量的生活实践经验当中通过归纳总结、演绎推理得出概念性知识和规律性知识的一个过程或者行为

二、口语化分类
1. 监督学习（类比 小初高，父母老师监督）
2. 无监督学习（类比 大学，自己激励管理自己）
3. 强化学习（类比 社会，遵纪守法，适应社会）


三、常用算法：
（1）. 监督学习（样本有类别标记）
	1. 分类（目标概念为已知类别）
		a. KNN（Nearest Neighbors Classification）（k-最邻近）
		b. DT（Decision Tree）（决策树）
		--预判信用等级
		c. SVM（Support Vectors Machine）（支持向量机）
		d. NN（Neural network）（神经网络）
		e. NB（Naive Bayes）（朴素贝叶斯）
	2. 回归（目标概念为连续值）
		a. LR（Linear regression）（线性回归）
		b. NLR（Non-Linear regression）（非线性回归）
（2）. 无监督学习（样本无类别标记）
	K-Means算法（一种聚类算法）

四、Python常用库
1. NumPy-python科学计算库
2. Pandas-python数据分析处理库
3. Scikit_learn-python机器学习库

五、SkLearn六大板块
1. 分类
2. 回归
3. 聚类
4. 维数约简
5. 特征抽取选择
6. 数据预处理

六、机器学习基本步骤：
1. 获取数据（获取、清洗、存储（hdfs、Hbase。。。））
2. 数据拆分训练集合测试集
3. 用特征向量训练算法
4. 在测试集上评估算法
5. 算法迭代与改进
6. 实际应用
7. 获取更多数据
8. 。。。

七、python开发步骤：
1. 导入数据集
2. 划分数据集
3. 模型实例化
4. 训练分类器
5. 使用分类器预测测试数据
6. 计算测试数据上的预测正确率

八、KNN：
1. 用未知和已知进行比对
2. 选择参数K
3. 根据投票法则归类

九、算法评估标准：
1. 准确率
2. 速度
3. 规模

3. 自动驾驶
4. 计算机视觉（人脸识别）
